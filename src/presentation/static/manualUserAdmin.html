Construção

Aqui você pode visualizar e gerenciar informações relacionadas à construção.

Botões disponíveis: Visualizar Informações, Adicionar, Editar, Excluir.

Exemplo de como adicionar uma nova construção:

Passos:

  1. Clique no botão "Gerenciar construções" no Menu.
  2. Clique no botão "Adicionar".
  3. Preencha corretamente as informações no formulário abaixo.
  4. Clique em "Salvar" para criar a alocação para um colaborador.

Exemplo de como editar uma construção:

Passos:

  1. Clique no botão "Gerenciar construções" no Menu.
  2. Visualize as informações que estão no momento para cada construção e ao final da linha estará o botão de Editar se estiver disponível para edição.
  3. abrirá um modal com as informações atuais da construção para edição.
  4. Clique em "Salvar" para salvar a edição.

Exemplo de como excluir uma construção:

Passos:

  1. Clique no botão "Gerenciar construções" no Menu.
  2. Visualize as informações que estão no momento para cada construção e ao final da linha estará o botão de Excluir.
  3. Abrirá um modal para confirmar a exclusão.
  4. Clique em "Excluir" para excluir.
